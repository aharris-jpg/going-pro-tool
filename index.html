How We Built This Tool What comes from the underlying school and state datasets, and what this calculator adds. +

This tool starts with school-level progression rates by sport. Each school’s baseline probability reflects historical pro output relative to its estimated athlete pool.

The page then compares that school baseline with the state average and national average for the same sport, where sport-specific state data is available.

To make the tool more interactive, it also applies a light profile adjustment based on year, playing level, and team tier. These are directional estimate refinements, not guarantees.

In simple terms, the model is: School Probability × Year Multiplier × Playing Level Multiplier × Team Tier Multiplier, with the final result capped so the profile layer cannot overwhelm the school baseline.
